Medium Components
=================
Medium components and general information.


## Parameters:

(* = required)

| Name                   | Description                         |
|------------------------|-------------------------------------|
| *data-stanza-medium-id | Medium identifier (e.g., NBRC_M231) |

## Sample:

```html
<div data-stanza="http://stanza-dev.herokuapp.com/stanza/medium_components" data-stanza-medium-id="NBRC_M231"></div>
```

The above `<div>` will automatically embed the following Stanza in your HTML page.

<div data-stanza="/stanza/medium_components" data-stanza-medium-id="NBRC_M231"></div>
